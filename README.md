# Real-Time Cryptocurrency Dashboard

Welcome to the Real-Time Cryptocurrency Dashboard! This project provides an interactive web application for visualizing and analyzing cryptocurrency market data. The dashboard features real-time updates, financial insights, and advanced charting to enhance your cryptocurrency tracking experience.

## Features

- **Real-Time Data Display**: Visualize live market data for the top cryptocurrencies, including BTC and ETH.
- **Dynamic Charts**: View financial insights through interactive charts, including candlestick charts and moving averages.
- **Automatic Data Updates**: Fetch and display the latest cryptocurrency data with automatic updates.
- **User Interface**: A responsive web interface that offers a clean and intuitive user experience.

## Tools and Technologies

- **Flask**: A micro web framework used to build the web application.
- **Plotly**: A graphing library for creating interactive and visually appealing financial charts.
- **yfinance**: A library for retrieving historical market data.
- **Requests**: For fetching data from the CoinMarketCap API.
- **HTML/CSS/JavaScript**: Technologies used for the front-end interface and dynamic content updates.

## Data Files

- **crypto_data.json**: Stores the latest cryptocurrency data fetched from the CoinMarketCap API.
- **index.html**: The main HTML file that renders the interactive charts and data display.
